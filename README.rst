==============================================================================
collective.editablemenu
==============================================================================

An editable menu for Plone 4 and 5

Installation
------------

Install collective.editablemenu by adding it to your buildout::

   [buildout]

    ...

    eggs =
        collective.editablemenu


and then running "bin/buildout"
